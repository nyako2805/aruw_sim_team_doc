Refill17MMZone

The refill 17mm zone class refills the robots using 17mm ammunition at a set rate, based off of the distance of the robot from the zone.

Fields

RefillInterval

int

The amount of time the tank has to wait before being refilled. Set to 1000 milliseconds by default.

RefillAmount

int

The amount of ammo added to the tank, once refilled. Set to 10 bullets by default.

team

int

The assigned team of the refill zone.

Methods

Start()

Is called when the game begins. Activates the refill zone.

Update()

Refills the robots using 17MM bullets surrounding the refill zone. This method is called every second.
